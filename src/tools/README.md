# Tools

このディレクトリは、**AI (LLM) が自律的に呼び出すためのツール (MCP サーバー等)** を配置する場所です。

## 役割と定義

- **AI 専用インターフェース**: AI が `tools` として認識し、自律的に情報取得や操作を行うためのコンポーネントのみを配置します。
- **MCP サーバー**: 主に Model Context Protocol (MCP) に準拠したサーバープログラムを格納します。

## 配置ルール

- ✅ **MCP サーバー**: 天気予報 (`weather`)、知識検索、外部 API 連携ツールなど。
- ❌ **システム・インフラツール**: ヘルスチェック用プロキシ (`health-proxy`)、監視ツール、デプロイ用スクリプトなどは `src/` 直下や `scripts/` に配置してください。
- ❌ **Body 直接操作**: 発話や感情変更などの「肉体」操作は `src/body` の REST API 経由で行うべきものであり、`src/tools` には配置しません。

## ディレクトリ構成

- `weather/`: 天気予報取得 MCP サーバー
